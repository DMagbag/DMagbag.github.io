PORTFOLIO: MY CURRENT PROJECTS

Virtual Pet

A virtual pet application that allows a user to adopt a dragon. The user's job is to care for this pet by selecting various activities, such as feeding it, giving it water, and playing with it. These activities affect different attributes including health level, energy level, hunger level, and boredom level. If levels drop to 0, the user failed to properly care for their pet and the game is over.

Virtual Pet Shelter

This application simulates the activities performed by a shelter worker, the user, to care for several animals. The user has the ability to give a pet up for adoption, admit new pets, and can interact with each pet individually. The user choices affect the attributes of each pet in the shelter such as health, where if you let a pet's health go to 0, the game is over.

Skills
